# Mapping the 18th-Century London Theatrical Community: The Newcomers (DHSI 2026)

## About
This small project was completed as a final project for the 
[Practical Javascript for Interactive Scholarship](https://dhsi.org/2025/09/25/practical-javascript-for-interactive-scholarship-dhsi-2026/) course at the 2026 session of the Digital Humanities Summer Institute. 

After the scripted portion of the workshop, students were required to create a small data-driven web project on a topic/using research data of their choice.

Completed in about 8 hours, with a few
adjustments applied after the
conference.

## Requirements 
* A Python virtual environment with the `dotenv` library
* A MapTiler API key

To deploy locally, you will need to populate a `.env` file with a valid
[MapTiler](https://www.maptiler.com/) API key. A free account is sufficient
to query and serve the basemap used for this project.


## Data
The site relies on lightly standardized personography data from
[Theatronomics](https://data.theatronomics.com/people), filtered and converted into JSON format.
